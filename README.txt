# 1. Abstract 

Joystick Component using SFML

About SFML, visit http://www.sfml-dev.org


# 2. How to build

# mkdir build 
# cd build
# cmake ../
# make


!! Important !!
Set environmental variable named SFML_ROOT for SFML root folder.
If SFML_ROOT is properly set, you can find %SFML_ROOT%include/SFML directory.
